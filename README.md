# Customer Behavior Analysis

## Overview
This project analyzes customer interaction data from an e-commerce platform to understand user behavior and engagement patterns.

## Tools
- Python
- Pandas
- Matplotlib

## Key Insights
- Most interactions are views, with low purchase conversion
- Peak activity occurs at specific hours
- User behavior varies across days

## Project Structure
- data/: dataset
- notebook/: analysis notebook
- results/: visualizations
